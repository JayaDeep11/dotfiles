# Arch Linux

#### Install Git

> before insallation it will update the system
```
sudo pacman -Syu git 
```

#### Install Packages

```
sudo pacman -S hyprland swww sddm kitty neovim neofetch 
```

#### Clone repo
```
git clone git@github.com:JayaDeep11/dotfiles.git
```
